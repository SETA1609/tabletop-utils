Welcome to Tabletop Utils!

This tool helps GMs track initiative in RPGs like D&D. Start with the Initiative Tracker in the navbar.

How to Use:

  • Add characters with names and initiative rolls.
  • The list auto-sorts by position (GM reorder) then initiative.
  • Click "Next Turn" to advance.
  • Delete as needed.

More docs coming. Check GitHub for source. Use the navbar for navigation, language/theme selectors.
